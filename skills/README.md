# Skills Directory

This directory contains custom AI Agent skills.

## Structure

Each skill should follow this structure:

```
skill-name/
├── SKILL.md              # Skill documentation
├── scripts/              # Executable scripts
│   ├── main.py
│   └── ...
├── references/           # Reference files
│   ├── config.json
│   └── ...
└── README.md            # Additional documentation
```

## Adding a New Skill

1. Create a new directory with your skill name
2. Add a `SKILL.md` file with skill documentation
3. Add scripts and references as needed
4. Update the main README.md with your skill

## Skill Template

See the OpenClaw documentation for the complete SKILL.md template:
https://docs.openclaw.ai/skills/authoring

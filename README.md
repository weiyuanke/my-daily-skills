# My Daily Skills

A collection of custom AI Agent skills for OpenClaw and other AI platforms.

## 📚 Skills

This repository contains reusable skills and tools for:
- OpenClaw personal AI assistant
- Custom AI workflows
- Agent automation

## 🚀 Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/weiyuanke/my-daily-skills.git

# Copy skills to your OpenClaw workspace
cp -r my-daily-skills/skills/* ~/.qclaw/workspace/skills/
```

### Usage

Each skill is self-contained and can be used independently. Refer to individual skill documentation for usage instructions.
